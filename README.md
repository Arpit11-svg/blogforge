# вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/Arpit11-svg/blogforge.git
```

Move into project

```bash
cd blogforge
```

Install dependencies

```bash
npm install
```

Create environment file

```bash
cp .env.sample .env
```

Start development server

```bash
npm run dev
```
